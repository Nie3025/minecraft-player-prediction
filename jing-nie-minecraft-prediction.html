1. Data Description

For this project, I use two related datasets: players.csv and sessions.csv, which record how 196 players interacted with the Minecraft research server over 1,535 sessions.

The players dataset includes demographic and behavioral information such as age, gender, experience level, and total playtime. The sessions dataset contains detailed records of each game session, including start and end times. Together, they provide both player-level and session-level perspectives on user engagement.

players.csv

Variable Type Description
experience Character Self-reported experience level (e.g., Pro, Veteran, Amateur).
subscribe Logical TRUE if subscribed to the newsletter, FALSE otherwise.
hashedEmail Character Anonymous ID used to connect player and session data.
played_hours Numeric Total hours spent on the server.
name Character Players in-game display name.
gender Character Players reported gender.
age Numeric Players age in years.

sessions.csv

Variable Type Description
hashedEmail Character Links each session to a player.
start_time Character Start time of the session.
end_time Character End time of the session.
original_start_time Numeric Start time as a Unix timestamp.
original_end_time Numeric End time as a Unix timestamp.

Most players did not subscribe to the newsletter, and a few played much longer than others, which may indicate outliers.
Since participation in the study was voluntary, the data might not represent the entire Minecraft player population. Some values, such as age or experience level, could be inaccurate or missing because they are self-reported. Time-related variables may also have inconsistencies due to time zone differences or logging delays.
Even with these limitations, the dataset still offers valuable information for understanding patterns related to newsletter subscription.

2. Questions

Broad question

What player characteristics and behaviors are most predictive of subscribing to a game-related newsletter?

Specific question

Can the number of sessions, average session duration, and total playtime predict whether a player subscribes to the newsletter?

Explanation

This project explores which types of players are more likely to subscribe to the Minecraft research newsletter.
Using players.csv, I will examine how age, experience level, gender, and total playtime relate to subscription status.
The response variable is subscribe (TRUE/FALSE), and the predictors are demographic and behavioral factors.
The goal is to identify engagement patterns that could help the UBC research team better target active players and plan future recruitment efforts.

4. Methods and Plan

I plan to use a logistic regression model to predict whether a player subscribes to the newsletter. This method is suitable because the response variable (subscribe) is binary (TRUE/FALSE). Other models such as decision trees could also be explored later, but logistic regression is chosen here because it offers clear interpretability and works well with small datasets.

Before modeling, I will check for missing values and ensure categorical variables are correctly formatted as factors. I will split the data into a training set (70%) and a test set (30%) to evaluate model performance. Accuracy and confusion matrix results will be used to assess prediction quality. Logistic regression is appropriate for this question because it is interpretable and helps identify which variables most strongly influence subscription behavior.
